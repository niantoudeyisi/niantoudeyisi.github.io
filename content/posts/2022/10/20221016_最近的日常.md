## 周四

遇到了个坑。代码写的少的坑。
单行if内部语句块一般不加括号。当注释掉里面的内容就会导致，后面的内容是是if里面的语句。如果后面也是一个单行if 就会出现蜜汁bug

```c
// 这就会导致第二个if都是第一个if里面的。。。

if (xxx)
//printf();
if (xxxx)
{
xxxxxx
}
```
